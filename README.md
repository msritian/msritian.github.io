# Shivam Mittal — Personal Site

Two-column, left-sticky sidebar portfolio. Content is JSON-driven. No build step.

## Local preview
- Open `index.html` in your browser.

## Update content
- Edit JSON in `data/`:
  - `profile.json` — name, tagline, bio, links, contact
  - `about.json` — location, resume link
  - `timeline.json` — work/experience timeline
  - `education.json` — education history
  - `skills.json` — grouped skills
  - `projects.json` — projects and achievements
  - `works.json` — projects with tags and years
  - `reading.json` — ongoing readings with status and notes

## Deploy on GitHub Pages

User Site (this repo must be named `msritian.github.io`). GitHub serves from root.
Alternatively, use the included GitHub Actions workflow; push to `main` to deploy.
